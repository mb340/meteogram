# Weather Meteogram

KDE Plasmoid showing weather forecast charts.

Forked from `blackadderkate/weather-widget-2` which was forked from `kotelnik/plasma-applet-weather-widget`.


### Requirements

 - KDE Plasma 5


### Data sources

 - The Norwegian Meteorological Institute
 - OpenWeather
 - Open-Meteo
 - sunrise-sunset.org
